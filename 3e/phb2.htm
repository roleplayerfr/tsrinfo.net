MAKE YOUR CHARACTERS THE BEST THEY CAN BE

Whether you're creating a new character or improving an existing one, the Player's Handbook II supplement has something you need. This volume provides new character options for players, including:

  • New class features for all of your favorite classes
  • A rebuilding system that allows you to update existing characters
  • New advancement options and affiliations
  • New starting equipment packages
  • New teamwork benefits
  • New feats and spells
