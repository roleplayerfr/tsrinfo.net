The d20 Modern Roleplaying Game core rulebook allows gamers to explore every modern fantasy roleplaying genre imaginable. Designed as a comprehensive stand-alone product, it also serves as the cornerstone for an array of support materials being created by a list of notable publishers. The flexible rules system gives long-time roleplayers the highly expandable, customizable modern roleplaying game they've been waiting for and provides new gamers with the perfct place to enter the world of modern roleplaying.
